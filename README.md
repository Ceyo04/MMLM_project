# 多模态大作业
计算机科学与技术专业 多模态大模型原理与应用课程 大作业

期末考核以课程综合项目报告为依据，包括项目设计、模型实现思路、结果分析、报告和最终展示

**期末验收**：第16-17周，提交最终报告（10-15页）+ 上台展示，包含：

* 系统实现细节与代码仓库
* 定量实验结果（至少1-2项指标）+ 可视化/错误分析
* 反思与展望（模型局限、AGI视角下的潜力与瓶颈）

**评分标准**：

| 评分项   | 权重 | 说明                               |
| -------- | ---- | ---------------------------------- |
| 开题报告 | 20%  | 调研深度、方案可行性、工作量规划   |
| 系统实现 | 30%  | 代码质量、功能完整性、创新性       |
| 实验评测 | 20%  | 实验设计合理性、指标选择、结果分析 |
| 最终汇报 | 30%  | Demo演示效果、报告清晰度、团队协作 |
